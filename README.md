# Lumi — карманный друг

Полноценный виртуальный питомец на Vite + React + TypeScript. Проект переработан из одностраничного прототипа `Pixel Pet`.

## Запуск

```bash
npm install
npm run dev
```

Для ускоренной проверки взросления и потребностей откройте адрес с `?fast=1`.

## Управление

- `A` / `←` — открыть меню и листать.
- `B` / `Enter` — выбрать.
- `C` / `Esc` — назад.
- Все действия также доступны мышью и на сенсорном экране.

## Что есть в игре

- яйцо и несколько этапов взросления с разными формами;
- выбор одного из четырёх семейств: росток, звёздный кот, аксолотль или облачный дракон;
- сытость, радость, здоровье, энергия, чистота и воспитание;
- сон, ночник, болезни, уборка, кормление и мини-игра;
- просьбы о внимании и влияние качества ухода на эволюцию;
- автоматическое сохранение и офлайн-симуляция до 72 часов;
- живой заголовок вкладки и favicon, которые показывают текущее состояние питомца;
- приветственная реакция питомца при возвращении во вкладку в уютном режиме;
- собственные доступные диалоги переименования и сброса вместо системных окон браузера;
- адаптивная игровая оболочка для компьютера и телефона;
- сгенерированные специально для проекта фон, корпус, питомцы и набор иконок.

## Визуальные режимы

- **Ретро** — включён по умолчанию: без боковых панелей и растровых ассетов. Корпус строится интерфейсом, а питомец и восемь иконок рисуются в `canvas` напрямую из пиксельных матриц.
- **Уют** — исходная иллюстрированная оболочка со сгенерированными ассетами и расширенными панелями состояния.

При создании нового яйца вид выбирается на экране самого устройства кнопками A/B или прямым нажатием. У каждого семейства есть собственные варианты яйца, малыша, подростка, взрослого, старичка и призрака.

Режим переключается кнопкой в правой части верхней панели и запоминается между запусками.

## Ассеты

Изображения находятся в `public/assets`. Они сгенерированы для этого проекта и подключаются как растровые PNG-ассеты; CSS отвечает за раскладку, а интерфейсные пиктограммы подключаются из `lucide-react`.
