# Живая мандрагора

Полноценный виртуальный питомец на Vite + React + TypeScript. Проект переработан из одностраничного прототипа `Pixel Pet`.

## Запуск

```bash
npm install
npm run dev
```

Для ускоренной проверки взросления и потребностей откройте адрес с `?fast=1`.

## Форматирование

```bash
npm run format
npm run format:check
```

## Управление

- `A` / `←` — открыть меню и листать.
- `B` / `Enter` — выбрать.
- `C` / `Esc` — назад.
- Все действия также доступны мышью и на сенсорном экране.

## Что есть в игре

- живой корень мандрагоры и несколько этапов взросления: от корневого кокона до защитного талисмана;
- одна выразительная героиня-альрауна с язвительным нравом, хорошей и запущенной взрослыми формами;
- сытость, радость, здоровье, энергия, чистота и воспитание;
- сон, ночник, болезни, уборка, кормление и мини-игра;
- отдельные спящие позы мандрагоры для каждой стадии;
- короткие реакции на кормление, игру, мытьё, лечение, сон и воспитание с наглядным изменением показателей;
- прямое общение с мандрагорой по нажатию и редкие покадровые idle-жесты, зависящие от её состояния;
- контекстные статусы, советы Герды и тематические реплики после действий;
- просьбы о внимании и влияние качества ухода на эволюцию;
- автоматическое сохранение и офлайн-симуляция до 72 часов;
- краткое стартовое окно при первом входе с объяснением офлайн-заботы и управления;
- живой заголовок вкладки и favicon, которые показывают текущее состояние питомца;
- приветственная реакция питомца при возвращении во вкладку;
- собственные доступные диалоги переименования и сброса вместо системных окон браузера;
- адаптивная игровая оболочка для компьютера и телефона;
- цельное лесное убежище: корневая колыбель, резной атлас состояния, ритуальная полка и атмосферное янтарное освещение;
- окружение, которое откликается на сон, выключенный свет, болезнь и новые просьбы питомца;
- сгенерированные специально для проекта интерьер жилища в корнях Госпожи Грюневальды и атласы мандрагоры.

## Внешний вид

Мандрагора живёт в корневой колыбели у очага, показатели оформлены как атлас живого корня, а действия — как ритуальные резные знаки.

При первом запуске мандрагора выбирается на экране самого устройства кнопкой B или прямым нажатием. Кнопки создания нового корня после подтверждения сразу запускают новую мандрагору без повторного экрана выбора. Её атлас включает корневой кокон, молодые стадии, две зависящие от ухода взрослые формы, древний талисман и дух альрауны.

## Ассеты

Изображения находятся в `public/assets`. Фон, корневая колыбель, физические кнопки и атлас ритуальных знаков хранятся в WebP. Обычный и спящий прозрачные атласы мандрагоры и favicon хранятся в PNG. Верхняя и боковые панели строятся на CSS, чтобы текст и интерактивные элементы аккуратно вписывались в них при любом размере экрана.
